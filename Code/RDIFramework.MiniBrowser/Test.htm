These are a few simple tests to test the functionality of the browser

If you allready see a pop-up, this can be because of the fact that you have the pop-up blocker disabled, or are running on Windows 2000 or lower.

Script error testing

  • When you load this document, you should also encounter a script error. This is on purpose.
  • Hit F5 to see if the script errors are still caught after a refresh
  • Click "Create an error" to test script error functionality

Closing windows

With this button you can try to close the current window.

Pop-up blocker

  • A pop-up can be displayed when the document is loaded. If not, an alert window will notify you. This is the kind of pop-ups you want to avoid.
  • A pop-up window can be because of an user action. This can be done using the button below.
  • When you click on the button, and you get the message that the pop-up is blocked, see if you have the filter level set to high. When this is the case, use the Ctrl Key to override the pop-up blocker
